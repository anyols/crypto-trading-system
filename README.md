# Crypto Trading System

A Python-based crypto market data and strategy research project.

## Current Features

- Connects to Coinbase Advanced Trade public WebSocket
- Streams real-time ticker data for BTC, ETH, SOL, DOGE, and XRP
- Parses live ticker messages into normalized Python objects
- Handles reconnects and graceful shutdowns

## Planned Features

- CSV and SQLite storage
- Historical OHLCV ingestion
- Donchian breakout strategy
- ATR-based risk management
- Backtesting and walk-forward validation
- TradingView Pine Script implementation
